## Quick Start & Documentation

### Requirements
Install the Angular CLI globally
```console
npm install -g @angular/cli
```

### Initial setup
cd into the local repo directory and run:
```console
npm install
```

### Running the application
```console
ng serve
```

### Running tests for the application
```console
ng test
```
